1. Política de Privacidade de Dados (Para Clientes da Eurocar)
Este documento serve para você (Oficina/Empresa) apresentar aos seus clientes, explicando como o sistema trata os dados deles.

Última atualização: 01 de Janeiro de 2026

A EUROCAR (Razão Social: Vitaliano Pereira Serpa, CNPJ: 59.152.856/0001-25) preza pela segurança dos dados de seus clientes. Esta política explica como coletamos e armazenamos informações através do nosso sistema interno de orçamentos.

1. Quais dados coletamos?
Para a emissão de orçamentos e ordens de serviço, nosso sistema coleta os seguintes dados pessoais e do veículo:

Dados do Cliente: Nome completo e Telefone de contato.

Dados do Veículo: Modelo, Marca e Placa.

2. Finalidade dos Dados
Os dados coletados têm como única finalidade:

Identificação do cliente e do veículo para formalização de orçamentos.

Contato para aprovação de serviços ou avisos de conclusão.

Registro histórico de manutenções realizadas no veículo.

3. Armazenamento e Segurança
Diferente de sistemas em nuvem, o sistema Eurocar opera em ambiente local.

Armazenamento Offline: Os dados dos clientes ficam armazenados exclusivamente no computador da empresa e em backups locais (arquivos .json e .pdf).

Não Compartilhamento: O software não transmite dados de clientes para servidores externos, terceiros ou nuvem pública. A conexão de internet do sistema é utilizada estritamente para verificar atualizações de versão do próprio software.

4. Direitos do Titular (LGPD)
O cliente tem o direito de solicitar à Eurocar, a qualquer momento:

Acesso aos dados cadastrados.

Correção de dados incompletos ou desatualizados.

Exclusão dos dados de nosso registro local (exceto quando a manutenção for necessária para cumprimento de obrigação legal/fiscal).

2. Termos de Uso do Software (Licença de Uso Interno)
Este documento é para quem opera o sistema (funcionários ou você mesmo), definindo responsabilidades sobre o uso do software.

1. Objeto
O Sistema Eurocar é uma ferramenta de propriedade intelectual privada, desenvolvida para gestão de orçamentos automotivos, geração de PDFs e histórico de serviços.

2. Responsabilidades do Operador
Ao utilizar o sistema, o usuário/operador concorda que:

Inserção de Dados: É responsável pela veracidade dos dados (Nome, Placa, Valores) inseridos nos campos de cadastro.

Sigilo: Não deve compartilhar os arquivos PDF ou JSON gerados com terceiros não autorizados, protegendo a privacidade dos clientes da oficina.

3. Backup e Armazenamento de Arquivos
O sistema salva os orçamentos em pastas locais configuradas pelo usuário (Padrão: C:\Users\Nome\Eurocar).

Backup: O desenvolvedor do software não possui cópia dos dados. É de inteira responsabilidade do usuário realizar backups periódicos das pastas de "Orçamentos PDF" e "Orçamentos Editáveis".

Em caso de formatação do computador ou falha no disco rígido, os dados poderão ser perdidos se não houver backup externo.

4. Atualizações
O sistema possui uma funcionalidade de verificação automática de atualizações via Google Drive.

O usuário concorda em manter o software atualizado para garantir a correção de eventuais bugs e a segurança da aplicação.

5. Propriedade Intelectual
Todo o código-fonte, design visual e lógica de geração de documentos pertencem exclusivamente aos detentores dos direitos autorais do Eurocar. É proibida a engenharia reversa, venda ou distribuição não autorizada deste software.
